#   Topic: OOP-02: Class and Object
1) Real World:
    Car:    Properties      Actions
            Color           Start
            Speed           Stop

2)  What is a Class?
    A class is a blueprint or template used to create objects.

3)  What is an Object?
    An object is an instance of a class.

4)  Difference Between Class and Object?
    Class	                                Object
    Blueprint	                            Actual instance
    No memory                               allocated	Memory allocated
    Defines properties and behavior	        Uses those properties and behaviors

    Example:
    Car design = Class
    Actual car = Object

Ex
    class Car
    {
    public:
        string color;
    };
